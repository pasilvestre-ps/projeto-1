1. História dos computadores

Voltar ao topo

       Os primeiros computadores foram desenvolvidos no final do século XIX e início do século XX, com marcos como a máquina analítica concebida por Charles Babbage em 1837.

       No entanto, os primeiros computadores eletrônicos programáveis surgiram durante a Segunda Guerra Mundial, com o Colossus na Grã-Bretanha e o ENIAC nos Estados Unidos. Após a guerra, houve avanços significativos, como o UNIVAC I, o primeiro computador comercialmente disponível, lançado em 1951.

       Desde então, houve uma rápida evolução, incluindo o desenvolvimento dos computadores pessoais nas décadas de 1970 e 1980.

2. O que são Clients?

Voltar ao topo

       Em tecnologia da informação, clients (clientes) se refere a dispositivos de computação ou softwares que acessam e utilizam os serviços, recursos ou dados fornecidos por um servidor. Esses clientes podem ser computadores pessoais, laptops, smartphones, tablets ou qualquer outro dispositivo capaz de se conectar a uma rede, como a internet, e interagir com servidores para acessar informações, enviar solicitações ou executar tarefas específicas.

       Os clientes podem se comunicar com os servidores por meio de diferentes protocolos de comunicação, como HTTP, FTP, SSH, entre outros. Em resumo, os clientes são os usuários ou sistemas que consomem os serviços disponibilizados por servidores em uma rede.

3. O que são Servers?

Voltar ao topo

       Servers são computadores ou sistemas de computação que fornecem serviços, recursos ou dados para outros dispositivos, chamados clients, em uma rede. Eles atuam respondendo a solicitações dos clientes, processando dados e fornecendo acesso a recursos compartilhados, como páginas da web, arquivos, e-mails, entre outros.

       Os servidores podem assumir diferentes funções, como servidores web, servidores de e-mail, servidores de banco de dados, entre outros, dependendo das necessidades e serviços oferecidos em uma rede. Em resumo, os servidores são responsáveis por fornecer e gerenciar informações e serviços para os clientes.

4. Como é uma estrutura básica de um HTML?

Voltar ao topo

       A estrutura básica de um documento HTML consiste em elementos que definem a estrutura e o conteúdo da página da web. Aqui está uma estrutura HTML simples:

Estrutura básica de um html

       Explicação dos elementos:

  • <!DOCTYPE html>: Define o tipo de documento HTML.
  • <html> Define o elemento raiz do documento HTML.
  • <lang="pt-br">: Atributo que define o idioma do documento.
  • <head>: Contém informações sobre o documento, como metadados, links para estilos e scripts.
  • <meta charset="UTF-8">: Define a codificação de caracteres do documento como UTF-8.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Define a largura da viewport e a escala inicial para dispositivos móveis.
  • <title>: Define o título da página exibido na barra de título do navegador.
  • <body>: Contém o conteúdo visível da página.
  • <h1>: Define um cabeçalho de nível 1.
  • <p>: Define um parágrafo de texto.

5. Tags

Voltar ao topo

       Em HTML, as tags são elementos usados para marcar partes específicas do conteúdo de uma página da web. Elas são cercadas por colchetes angulares < > e podem ter atributos que fornecem informações adicionais sobre como o elemento deve ser exibido ou comportar-se.

       As tags podem ser usadas para diversos fins, como definir cabeçalhos, parágrafos, links, imagens, tabelas, formulários e muito mais. Elas são fundamentais para estruturar e formatar o conteúdo de uma página da web, permitindo que os navegadores entendam como exibir e interpretar o documento HTML.

6. Atributos das Tags

Voltar ao topo

       Atributos das tags HTML são características adicionais que podem ser adicionadas a elementos HTML para fornecer informações extras ou para controlar o comportamento e a aparência desses elementos. Cada elemento HTML pode ter seus próprios atributos, e esses atributos são especificados dentro das tags de abertura dos elementos.

       Por exemplo, na tag <img>, o atributo src é usado para especificar o URL da imagem a ser exibida, enquanto o atributo alt é usado para fornecer um texto alternativo caso a imagem não possa ser exibida.

       Os atributos podem ser usados para diversas finalidades, como fornecer informações de formatação (como cores, tamanhos de fonte), definir comportamentos (como links para outras páginas, ações de formulário), fornecer metadados (como idioma do documento, codificação de caracteres), entre outros. Eles são uma parte fundamental na construção e na personalização de páginas da web.

7. Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar ao topo

       Listas ordenadas são usadas quando a ordem dos itens é relevante e precisam ser numeradas ou ordenadas, enquanto listas não ordenadas são utilizadas quando a ordem dos itens não é importante e apenas a apresentação de uma lista de itens é necessária.

8. HTML não é uma linguagem de programação.

Voltar ao topo

       O HTML é uma linguagem de marcação que define a estrutura e o conteúdo de uma página da web, enquanto linguagens de programação como Python JavaScript são usadas para adicionar interatividade e funcionalidade às páginas.

9. Links

Voltar ao topo

       A tag <link> em HTML é usada para conectar um documento HTML a recursos externos, como folhas de estilo CSS, ícones de favoritos e fontes. Ela não possui uma tag de fechamento.
